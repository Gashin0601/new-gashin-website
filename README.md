# 鈴木我信 ウェブサイト

個人ブランド・実績・物語を発信する次世代ウェブサイトプロジェクト

## プロジェクトステータス

**現在：コンセプト策定中**

## 特徴

- 最新のWeb技術（Next.js 15 + Tailwind CSS v4）
- シネマティックな演出（動画とコードの同期）
- 本人録音による音声ナレーションシステム
- アクセシビリティ最優先（Lighthouse A11y 100目標）
- パフォーマンス最適化（LCP < 2.0s目標）

## ドキュメント

詳細なプロジェクトコンセプトは [`CONCEPT.md`](./CONCEPT.md) を参照してください。

## 技術スタック

- **フレームワーク**: Next.js (App Router, RSC) + TypeScript
- **スタイリング**: Tailwind CSS v4
- **デプロイ**: Vercel
- **音声**: Web Audio API
- **動画**: ホスト動画 + YouTube no-cookie埋め込み
